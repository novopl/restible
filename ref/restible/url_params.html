restible.url_params¶

Functionality related to filters.

Filters are used by when querying the resource. For now the functionality is quite simple - we just extract the filters from GET query params.

restible.url_params.from_string(value)[source]¶

Convert the given string value to the actual type it holds.

If the string is an integer it will return an int, if it’s a float, then it will return float otherwise it will just return a string.
